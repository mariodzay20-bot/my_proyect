# Análisis de Datos de Vehículos - Aplicación Web

## Descripción del proyecto
Esta aplicación web permite explorar un conjunto de datos de anuncios de venta de coches mediante visualizaciones interactivas.

## Datos utilizados
El dataset contiene información sobre vehículos usados incluyendo precio, kilometraje, año, marca, modelo, etc.

## Funcionalidades
- Histograma interactivo del kilometraje de los vehículos
- Gráfico de dispersión precio vs kilometraje
- Interfaz intuitiva con botones para generar visualizaciones

## Acceso a la aplicación
La aplicación está desplegada en: https://my-proyect-1-imii.onrender.com

## Uso
1. Haz clic en "Construir histograma" para ver la distribución del kilometraje
2. Haz clic en "Construir gráfico de dispersión" para ver la relación precio-kilometraje
